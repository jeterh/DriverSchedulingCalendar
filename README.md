車隊排班管理系統 (Driver Shift Scheduler)

這是一個專為車隊管理設計的自動化排班工具，旨在簡化排班流程、確保法規合規性（如勞基法七休一），並提供直觀的視覺化介面供管理員進行手動調整與列印報表。

核心功能

自動排班演算法：

智慧分配「休假」與「機動代班」。

自動符合月休天數基準。

優先確保承攬人員於平日執勤，約聘人員優先輪休。

內建「七休一」邏輯，防止人員連續工作超過 6 天。

直觀互動介面：

點擊切換狀態：點擊儲存格即可在「出勤 → 休假 (✕) → 機動 (◇) → 特休 (特)」之間快速切換。

欄位鎖定：姓名與車號在橫向捲動時保持鎖定，方便大數據對照（手機版優化）。

資料持久化：自動將排班資料與人員設定儲存於瀏覽器本地空間 (LocalStorage)。

專業列印模式：針對 A4 橫式列印優化，自動隱藏操作按鈕並強化表格邊框。

狀態標示說明

為了提升閱讀速度，系統採用以下視覺化代碼：

休假 (✕)：紅色叉號，代表當日不排班。

機動 (◇)：藍色菱形，代表當日為機動待命或代班。

特休 (特)：紫色文字，代表人員請領特別休假。

空白：一般正常出勤。

技術架構

框架：React

樣式：Tailwind CSS

圖示：Lucide React

部署：單一檔案架構，支援即時預覽與靜態部署。

開發與修改

如需修改人員名單或車輛資料，可點擊右上角的「設定」齒輪圖示進入管理介面進行編輯。
