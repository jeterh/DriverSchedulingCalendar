Release Note - v1.0.0 (Initial Release)

版本摘要

這是「車隊排班管理系統」的第一個正式版本。本版本完成了核心排班邏輯的建構，並針對使用者回饋優化了介面顯示與行動裝置的體驗。

新增功能

[UI/UX] 視覺化狀態說明 (Legend)：

於月份選擇器右側新增狀態圖示說明（休假、機動、特休）。

採用簡約的膠囊設計，區分紅、藍、紫三種狀態顏色。

[RWD] 行動版佈局優化：

實現「姓名」與「車號」雙欄位左側鎖定，確保在手機小螢幕上捲動時仍能清楚辨識人員。

「車型」欄位調整為跟隨捲動，釋放更多可視空間。

[Feature] 車隊管理介面：

側邊選單式管理，支援即時新增、修改、刪除司機與車輛資訊。

[Print] 專業報表輸出：

優化印表機 CSS 樣式，自動帶入月份標題與圖示註解。

排班邏輯說明 (v1.0)

週末權重：週末目標排休人數預設為 8 人，平日為 3 人。

連作禁止：系統將自動阻斷連續工作超過 6 天的人員，並在第 7 天強制排休。

機動分配：機動班優先分配給「預設職責為機動」的人員，且同一人連續機動天數不得超過 5 天。

修復與優化

修正了車型與車號欄位順序。

調整表格高度，提升在觸控螢幕上的點擊精確度。

優化 LocalStorage 儲存結構，防止版本更新導致的資料損毀。
